# 📝 Lista de Tarefas em Python

Projeto simples de lista de tarefas desenvolvido em Python com o objetivo de praticar lógica de programação, funções, listas e manipulação de dados.

O projeto possui duas versões:
- 💻 Versão no terminal
- 🖼️ Versão com interface gráfica utilizando Tkinter
---
# 🚀 Funcionalidades

O sistema permite:

- ➕ Adicionar tarefas
- ✏️ Atualizar tarefas
- ❌ Remover tarefas
- 🚪 Sair do programa
---
# 🛠️ Tecnologias Utilizadas
- Python
- Tkinter (interface gráfica)
